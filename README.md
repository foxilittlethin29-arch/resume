# 🚀 Mehboob Khalid – Interactive Resume & Developer Portfolio

A modern, responsive, and interactive Resume & Portfolio website built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. This project showcases my professional experience, technical skills, featured projects, and software development journey through a clean and premium user interface.

Designed with performance, responsiveness, and modern UI/UX principles in mind, this application serves as my online developer profile for recruiters, clients, and potential collaborators.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/foxilittlethin29-arch/resume.git
```

Navigate to the project:

```bash
cd resume
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
