# AI Chatbot SaaS Project Images

Place your project screenshots in this folder.

## Expected Images

| File | Description | Required |
|------|-------------|----------|
| `cover.webp` | Main project thumbnail (1200x800) | Yes |
| `dashboard.webp` | Chat interface screenshot | Optional |
| `feature-1.webp` | Feature showcase 1 | Optional |
| `feature-2.webp` | Feature showcase 2 | Optional |
| `mobile.webp` | Mobile view screenshot | Optional |

## Image Guidelines

- Format: WebP (recommended) or JPEG
- Thumbnail size: 1200x800px
- Screenshot size: 1920x1080px
- Max file size: 500KB per image
